# 🚀 Hex Custom - Tunnel AIO

🔠🔠🔠🔠🔠🔠🔠🔠🔠🔠 🔠  

## ◻️ Aplicación VPN Customizable

**Hex Custom** es una **aplicación VPN totalmente configurable**, diseñada para que puedas **agregar, administrar y compartir tus propios servidores VPN** de forma sencilla, segura y flexible.

Está pensada para usuarios avanzados, pruebas de laboratorio y personas que necesitan una VPN tipo **HTTP Custom**, con múltiples métodos de exportación y compatibilidad con diversos protocolos.

---

## ✨ Características Principales

- ◻️ VPN **customizable**
- ◻️ Agrega **múltiples servidores**
- ◻️ Administra y reutiliza configuraciones fácilmente
- ◻️ Comparte servidores con diferentes métodos de exportación
- ◻️ Interfaz simple con opciones avanzadas

---

## 🌐 Protocolos Compatibles

Hex Custom soporta una **amplia variedad de protocolos** para adaptarse a distintas redes y necesidades:

- 💻 **SSH / SSL**
- 💻 **SSH / PAYLOAD**
- 💻 **SSH / PROXY**
- 💻 **SSL / PAYLOAD**
- 💻 **SSH / SLOWDNS**
- 💻 **UDP (ZIVPN | Hysteria 1)**
- 💻 **V2RAY / XRAY**
- 💻 **PSIPHON TUNNEL**

---

## 📱 Requisitos

- Android 7.0 (API 24) o superior
- Permiso para crear conexiones VPN
